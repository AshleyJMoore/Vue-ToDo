#Simple Vue JS 'ToDo' app.

- Add a new item to the list with priority
- Remove an item from the list
- Clear the entire list
- Stores list in local storage to persist after page reload

## Technologies
 - Vue JS
 - Vuex
 - ESLint + standard
 - Bootstap
 - SASS
